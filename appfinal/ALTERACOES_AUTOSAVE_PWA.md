# Alterações incluídas neste pacote

Este pacote já está pronto para subir no GitHub/Vercel e inclui:

- Supabase Auth, Database, Storage e Realtime.
- Dashboard com dados reais.
- Ferramentas com modelo, valor e data de compra.
- Melhorias de usabilidade e performance.
- Validações contra duplo clique e números negativos.
- PWA instalável pelo Chrome.
- Salvamento automático de rascunhos em formulários.

## Formulários com rascunho automático

Os dados digitados permanecem salvos mesmo que o usuário:

- troque de aba;
- atualize a página;
- feche e abra o navegador;
- saia no meio do cadastro.

O rascunho é limpo automaticamente após salvar com sucesso.

Formulários protegidos:

- Nova obra;
- Material;
- Ferramenta;
- Progresso físico;
- Checklist;
- Atividades.
