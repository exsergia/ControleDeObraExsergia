Exsergia Hub

Aplicação web/PWA para gestão operacional de obras, materiais, progresso físico, ferramentas, frota, documentos fiscais, financeiro, relatórios e scanner IA fiscal.

Visão Geral

O Exsergia Hub centraliza a operação de campo e a gestão administrativa em uma única plataforma conectada ao Supabase.

FrontendReact, TypeScript, Vite e Tailwind CSS.
BackendSupabase Auth, Database, Storage, Realtime e Edge Functions.
APIRotas serverless Vercel em /api/*.
IA FiscalScanner de imagem de NF/Cupom via Edge Function.

NF/Cupom Fiscal

  • Operador pode cadastrar NF/Cupom.
  • Operador visualiza somente documentos que lançou.
  • Administrador visualiza e edita todos.
  • Campo Despesas substitui o antigo fornecedor.
  • Despesas disponíveis: ALMOÇO, JANTA, CAFE, ESTACIONAMENTO, HOSPEDAGEM, MATERIAL.
  • Campo de despesa não é obrigatório.
  • Há campo de observação no lançamento.
  • Filtros por obra e pessoas envolvidas.

Scanner IA Fiscal

A Edge Function supabase/functions/analyze-fiscal-image/index.ts analisa imagens de NF/Cupom Fiscal e retorna status, confiança, valor, data, fornecedor/despesa provável, avisos e motivos.

O resultado fica salvo em fiscal_docs.data.aiAnalysis.

A imagem enviada ao modelo é processada como entrada visual e consome tokens. O app usa detalhe baixo para reduzir custo.

API Serverless

A API fica em /api/* e exige token Supabase nas rotas protegidas.

Authorization: Bearer <access_token_do_supabase>
MétodoRotaUso
GET/api/healthVerifica configuração.
GET/api/fiscal-docsLista NF/Cupom.
POST/api/fiscal-docsCria NF/Cupom.
PATCH/api/fiscal-docs?id=<id>Edita NF/Cupom.
GET/api/records?table=obrasLista registros permitidos.
POST/PUT/PATCH/api/records?table=fiscal_docsGrava registros permitidos.

PWA e Plataformas

O app é responsivo e preparado para desktop, Android, iPhone/iOS, tablets e instalação como PWA.

  • Layouts responsivos.
  • Inputs e botões adaptados para mobile.
  • Persistência de abas e formulários em partes críticas.
  • Service worker e manifest configurados.

Deploy e Validação

npm install
npm run lint
npm run build
  • Configurar variáveis no Vercel.
  • Configurar secrets da IA no Supabase.
  • Rodar migrations necessárias.
  • Validar login por e-mail, CPF e telefone.
  • Validar cadastro, upload de foto, NF/Cupom e scanner IA.
  • Confirmar que /api/* retorna JSON.

Cuidados de Produção

  • Não expor service role nem chave OpenAI.
  • Manter RLS ativo.
  • Não apagar dados sem backup.
  • Testar mobile/iPhone após alterações de auth, câmera ou upload.
  • Monitorar falhas de rede do Supabase.
